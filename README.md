# customer_behavior_analysis
data analytics project showcasing customer behavior analysis using python, sql and power bi
📊 Data Analytics Project
🔎 Overview

This project demonstrates an end-to-end data analytics workflow, starting from raw data processing to delivering actionable insights through visualization and reporting. The goal is to extract meaningful patterns, improve data quality, and present findings in a clear and business-friendly format.

📁 Dataset
The dataset used in this project contains structured data relevant to the problem domain.
It includes multiple features such as categorical and numerical variables.
Data quality issues like missing values, duplicates, and inconsistencies were handled during preprocessing.
🛠️ Tools & Technologies
Python – Data loading, cleaning, and exploratory data analysis (EDA)
MySQL – Querying and analyzing structured data
Power BI – Interactive dashboard creation
Gamma – Presentation (PPT) creation
Libraries – Pandas, NumPy, Matplotlib, Seaborn
⚙️ Project Steps
1. Data Loading
Imported dataset using Python
Verified structure, data types, and initial quality
2. Exploratory Data Analysis (EDA)
Analyzed distributions and relationships
Identified trends, correlations, and outliers
Visualized key insights using charts and graphs
3. Data Cleaning
Handled missing values
Removed duplicates
Standardized formats and corrected inconsistencies
4. SQL Analysis (MySQL)
Loaded cleaned data into MySQL
Wrote queries to:
Aggregate data
Filter key segments
Generate business insights
5. Dashboard Creation (Power BI)
Built interactive dashboards
Included KPIs, filters, and visual summaries
Designed for easy interpretation by stakeholders
6. Reporting
Compiled findings into a structured report
Highlighted key insights and recommendations
7. Presentation (Gamma)
Created a professional presentation
Focused on storytelling and clear communication of results
📊 Dashboards
Interactive Power BI dashboard showcasing:
Key performance indicators (KPIs)
Trends and comparisons
Drill-down analysis using filters
📈 Results
Improved data quality and consistency
Identified key patterns and trends in the dataset
Generated actionable insights to support decision-making
Delivered results through dashboards, reports, and presentation
